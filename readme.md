# Chatbot using LLM (Text + Voice)

This project is a mini project created by Harshvardhan Tripathi.  
It is a chatbot that supports:
- Text input
- Voice input (speech-to-text)
- Voice output (text-to-speech)
- LLM-based intelligent responses using Google Gemini API

## Technologies Used
- Python
- FastAPI
- Streamlit
- Gemini API (Google Generative AI)
- SpeechRecognition
- GTTS (Google Text-to-Speech)
- Pydub

## How to Run
1. Install Python 3.10+
2. Create virtual environment:python -m venv venv
3. Activate environment:venv\Scripts\activate
4. Install dependencies:pip install -r requirements.txt
5. Run the project:run.bat


## Features
- Chat using text  
- Speak using microphone  
- Hear AI reply in audio  
- Shows chat history  
- Dark mode UI
